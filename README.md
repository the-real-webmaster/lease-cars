# 🚗 Lease Cars Landing Page

Одностраничный адаптивный лендинг для автосалона. Демонстрирует выбор авто с табами, блог, видео, форму контактов и футер. Используются **CSS Grid**, **Flexbox**, кастомные шрифты и адаптивная верстка для разных экранов.

---

## 🔗 Ссылка
[Посмотреть проект](https://the-real-webmaster.github.io/lease-cars/)

---

## 🛠 Технологии
- HTML5  
- CSS3 (Flexbox, Grid, адаптивный дизайн)  
- JavaScript (для табов и интерактивных элементов)  
- Google Fonts (Inter)  

---

## 🔹 Особенности
- Интерактивные табы  
- Адаптивная сетка карточек авто  
- Секция блога с карточками  
- Встроенное видео  
- Форма контактов с валидацией  
- Полностью адаптивный дизайн для любых устройств  

---

## 📱 Адаптивность
Проект корректно отображается на различных экранах: десктоп, планшет, мобильные устройства.
